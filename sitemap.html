Информация о сайте

Технологии

Сайт разработан с использованием современных технологий:

  • Backend: Python Flask
  • Frontend: HTML5, CSS3, JavaScript
  • База данных: SQLite/PostgreSQL
  • Фреймворк: Bootstrap 5
  • Иконки: Font Awesome

Доступность

Сайт соответствует требованиям доступности:

  • ГОСТ Р 52872-2012 для слабовидящих
  • Адаптивный дизайн
  • Семантическая верстка
  • Поддержка скринридеров
  • Клавиатурная навигация

Обновления

  • Ежедневные новости
  • Еженедельные статьи
  • Обновление прайс-листа
  • Добавление новых услуг
